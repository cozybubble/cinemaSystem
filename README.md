# cinemaSystem
java大作业，在控制台运行的电影院管理系统
## 项目展示
###  主菜单界面：
```
--------------主菜单------------
1. 管理员登录
2. 用户登录
3. 显示所有影厅
4. 显示所有电影
5. 显示所有放映
6. 注册新用户
0. 退出
------------------------------
请输入操作序号: 
```
